# 1. 编辑环境变量
```
编辑.bashrc或者.profile，加入以下内容： 
export GOROOT=$HOME/go 
export GOBIN=$GOROOT/bin 
export GOARCH=amd64 
export GOOS=linux 
```

# 4. 源码安装
## 4.1 通过git安装
### 4.1.1 先安装go1.4
```
cd ~/resources
git clone https://github.com/golang/go.git
cd go
git checkout release-branch.go1.4
cd src
export GO14TESTS=1
export CGO_ENABLED=0
./make.bash
export GOROOT_BOOTSTRAP=$HOME/go1.4

cd ~/resource/go
git checkout master
cd src

```
